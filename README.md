# 🧭 WanderLust

**Airbnb-style travel rental platform — browse and list properties with maps & photo upload support.**

WanderLust is a full-stack web application inspired by Airbnb. Users can explore rental properties with location mapping (MapTiler) and upload property images using Cloudinary. Authentication enables users to register/login and create listings.  
🚫 **Booking / reservation functionality is not implemented.**

---

## 🚀 Features

- User authentication (Sign up / Login / Logout)
- Create, edit & delete property listings
- Upload property images using Cloudinary
- Interactive map integration with MapTiler
- View listings with location-based visualization
- Flash messages for user feedback
- Responsive UI using EJS templates

### Prerequisites

Ensure you have the following installed:

- Node.js (v16+)
- npm
- MongoDB (local or Atlas)
- Cloudinary account
- MapTiler API key
